1. 두개의정수n과m이주어집니다. 반복문을사용하지않고별(*) 문자를이용해가로의길이가n, 세로의길이가m인직사각형형태를출력해보세요.

```python
n = 5
m = 9
star = '*'
newline = '\n'
print(((star * n) + newline) * m )

garo = '*' * n
print(((garo + '\n') * (m - 1)) + garo)
```
