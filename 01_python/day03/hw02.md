2. range와 slicing을 활용하여 1부터 50까지 숫자 중 홀수로 이루어진
   리스트를 만드시오.

```python
#1
list_odd = []
for number in range(1, 51):
    if number % 2 == 1:
        list_odd.append(number)
print(list_odd)

#2
list_odd = list(range(1, 50, 2))
print(list_odd)
```
